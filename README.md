Airport Challenge
=================

Software to control the flow of planes at an airport.

The following classes are used.

### Class - Airport 
 Responisibilites        |Collaborators
 ------------------------|------------------
 Hold plane              |Plane
 Dispatch plane          |Plane
 Land plane 			 |Plane

### Class - Plane 
 Responisibilites        |Collaborators
 ------------------------|------------------
 Be at the airport       |Airport
